# PressHouse Enterprise Platform (بيت الصحافة - اليمن)

PressHouse is a comprehensive digital institution platform designed for managing media operations, journalist memberships, and human rights observatory tasks in Yemen.

## 🚀 Key Features
- **Unified Backend**: Single API gateway for all institutional modules.
- **Enterprise Architecture**: Modular monolith design with SQLite/MySQL support.
- **AI Integration**: Powered by Gemini for content analysis and user assistance.
- **Media Center**: Centralized asset management with local storage support.
- **Observatory & Legal**: Dedicated tools for tracking violations and legal cases.
- **Journalist Portal**: Specialized dashboards for registered journalists.

## 🛠️ Tech Stack
- **Frontend**: React 19, Tailwind CSS 4, Framer Motion, Lucide Icons.
- **Backend**: Node.js (tsx), Express.
- **Database**: SQLite (via `better-sqlite3`) with Drizzle-like migration logic.
- **Localization**: Full RTL/LTR support via `i18next`.
